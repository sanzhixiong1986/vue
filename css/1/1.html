重点：

  1. css是层叠样式表
  2. 使用link标签引用外部的css下面是例子,link必须放入到head里面
  3. 其中type说明加载数据的类型,href是样式表的路径，这里可以是绝对的路径和相对路径或者网上的网址
  4. 多个css合并成一个，减少http的请求
  5. 另一种方式使用style标签：可以直接在里面写样式，也可以用@import url导入css文件夹，这个和link使用方式一模一样，
  6. css 注释和其他的语言差不多，比如c++，java等 /**/
  7. css 内嵌方式就是在标签直接使用style然后在属性：数值见demo2
  8. 如果外部css和标签style比较同一种原色进行操作：内嵌>标签style>外部链接
